## Setup Guide

1. Power OFF Arduino
2. Complete wiring as per hardware tables
3. Install required libraries
4. Upload `MASTER_DEBUG_RFID.ino`
5. Verify firmware version in Serial Monitor
6. Upload `RFID_READ_NEW_CARD.ino`
7. Note down UID
8. Paste UID in `VERIFIED_REGISTERED_RFID.ino`
9. Upload final code
